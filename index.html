🏗️ Phase 1: Foundation (Server-side)

Master the fundamentals - server-side vulnerabilities are easier to learn and form the base of web security

📚 13 Topics 🧪 105+ Labs ⏱️ 2-3 Months

SQL Injection

Apprentice 🔥 Start Here Open Labs
18 Labs ⏱️ 2-3 weeks

Learn to exploit SQL injection vulnerabilities to access unauthorized data. The most critical vulnerability to master first.

Authentication

Apprentice 🔥 Critical Open Labs
14 Labs ⏱️ 2 weeks

Exploit authentication flaws including password brute forcing, broken brute-force protection, and multi-factor authentication issues.

Access Control Vulnerabilities

Apprentice 🔥 Critical Open Labs
13 Labs ⏱️ 2 weeks

Learn vertical and horizontal privilege escalation, IDOR vulnerabilities, and how to bypass access control mechanisms.

Path Traversal

Apprentice High Open Labs
6 Labs ⏱️ 1 week

Exploit insufficient validation of user-supplied file paths to access files outside the intended directory.

OS Command Injection

Apprentice High Open Labs
5 Labs ⏱️ 1 week

Inject operating system commands through vulnerable application code to execute arbitrary commands on the server.

File Upload Vulnerabilities

Apprentice High Open Labs
7 Labs ⏱️ 1-2 weeks

Exploit unrestricted file uploads to upload malicious files, including web shells and XSS payloads.

Server-side Request Forgery (SSRF)

Apprentice High Open Labs
7 Labs ⏱️ 1-2 weeks

Trick the server into making requests to unintended locations, potentially accessing internal resources.

XXE Injection

Practitioner Medium Open Labs
9 Labs ⏱️ 1-2 weeks

Exploit XML External Entity vulnerabilities to read files, perform SSRF, and more.

Information Disclosure

Apprentice Medium Open Labs
5 Labs ⏱️ 1 week

Find and exploit information leakage through error messages, verbose responses, and exposed data.

Business Logic Vulnerabilities

Practitioner Medium Open Labs
11 Labs ⏱️ 2 weeks

Exploit flaws in application workflow design that allow attackers to cause unexpected behavior.

API Testing

Practitioner Medium Open Labs
5 Labs ⏱️ 1 week

NoSQL Injection

Apprentice Medium Open Labs

Race Conditions

Practitioner Medium Open Labs
